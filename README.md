# Vibe Landing

A modern, responsive product landing page template with interactive features and smooth user experience.

## Description

A clean and professional single-page product landing template built with vanilla HTML, CSS, and JavaScript. Features a sticky navigation bar, smooth scrolling, interactive modals, and fully responsive design that works seamlessly on mobile and desktop devices.

## Features

### v1 Core Features
- **Hero Section** - Eye-catching hero with call-to-action buttons
- **Product Showcase** - Feature cards highlighting key product benefits
- **Benefits Section** - Clear value propositions with icons
- **Testimonials** - Social proof with customer quotes
- **Pricing Section** - Three-tier pricing cards with interactive buttons
- **Footer** - Social links and dynamic copyright year

### Enhanced Features (Merged)
- **Sticky Navigation** - Navigation bar stays visible while scrolling with smooth section links
- **Smooth Scrolling** - Seamless anchor link navigation throughout the page
- **Active Section Highlight** - Navbar automatically highlights the current section on scroll
- **CTA Modal** - "Get Started" button opens a modal with contact form
- **Form Validation** - Real-time validation for required fields and email format
- **Modal Controls** - Close modal via close button, backdrop click, or Escape key
- **Interactive Pricing** - Pricing buttons provide user feedback on selection
- **Responsive Design** - Mobile-first approach with breakpoints for all screen sizes

## How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/product-landing-AK.git
   cd product-landing-AK
   ```

2. **Open in browser**
   - Simply open `index.html` in your preferred web browser
   - Or use a local server for better development experience:
   
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Access the site**
   - Navigate to `http://localhost:8000` (or the port you specified)
   - Or double-click `index.html` to open directly in your browser
